      📤 Publish Menu

      Customers load the menu from menu.json on your server. After making changes, export and upload this file to your hosting (Netlify, GitHub Pages, cPanel, etc.).

      How it works:
      1. Edit your menu items and restaurant info above
      2. Click Export menu.json to download the file
      3. Upload menu.json to the same folder as index.html on your host
      4. Customers scanning your QR code will see the updated menu instantly
