## 🛠️ Customization Guide

The banner is easy to customize:

### Change Colors:
Edit `style.css`:
```css
.promo-banner {
  background: linear-gradient(135deg, #YOUR_COLOR1, #YOUR_COLOR2);
}
```

### Adjust Animation Speed:
```css
.promo-text-main {
  animation: float 6s ease-in-out infinite;  /* Change 6s */
}
```

### Add/Remove Elements:
Edit `index.html` promotional card section

See `PROMOTIONAL_CARD_GUIDE.md` for detailed customization options.
