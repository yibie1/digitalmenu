      � Auto-Publish to GitHub

      Connect your GitHub repo so every change you make here is instantly live for all customers — no manual uploading needed.

      ▶ How to get a GitHub Token (one-time setup)
      1. Go to github.com → Settings → Developer settings
      2. Click Personal access tokens → Tokens (classic)
      3. Click Generate new token (classic)
      4. Give it a name e.g. menu-publish
      5. Tick the repo checkbox (full repo access)
      6. Click Generate token and copy it
      7. Paste it below — it's stored only in your browser

      🖼 Image Hosting (Cloudinary)

      Without this, food photos are stored as base64 inside menu.json which causes the file to exceed GitHub's size limit and images won't sync to other devices. Cloudinary is free and fixes this permanently.

      ▶ Setup (2 minutes, free)
      1. Go to cloudinary.com → Sign up free
      2. In your dashboard, copy your Cloud Name
      3. Go to Settings → Upload → Upload presets
      4. Click Add upload preset → set Signing Mode to Unsigned → Save
      5. Copy the preset name and paste below
